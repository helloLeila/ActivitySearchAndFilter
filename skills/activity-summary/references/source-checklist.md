# Source Checklist

先搜这些最容易出真活动的源头：

## 第一优先级

1. 高校官网、学院官网、研究院官网、官方 event calendar
2. CCF、ACM、IEEE 官网或官方公众号
3. Luma、Meetup 官方活动页
4. 主办方官网

## 第二优先级

1. 腾讯、阿里、字节、华为技术社区
2. AWS 社区、OceanBase、Deepin、CANN
3. CNCF、K8s、Go、Rust、WebAssembly
4. LangChain、LLM 社区、PyTorch、TensorFlow、向量数据库、机器人社区

## 第三优先级

1. InfoQ、极客时间、SegmentFault、掘金、OSC、CSDN 线下活动
2. 前端早早聊、freeCodeCamp、OpenCloudOS、GitCode、开源之夏
3. 大湾区 AI、前端、Agent、具身智能社区

## 搜索顺序建议

1. 先搜深圳高校和深圳学术组织
2. 再搜香港高校与香港高含金量活动
3. 再补广州、珠海、澳门
4. 最后用开发者社区补小型 meetup

## 核验优先级

同一活动多平台出现时，优先保留：

1. 高校官网 / 学院官网 / 官方日历
2. 官方公众号原文
3. 主办方官网
4. Luma / Meetup
5. 媒体转发页

媒体稿、二手转发页只能辅助验证，不能作为唯一主链接。
