# Output Template

每次输出都按下面结构，不要自由发挥。

## 1. 检索结果

- 时间范围：YYYY-MM-DD 00:00 至 YYYY-MM-DD 23:59
- 本次找到：X 场符合条件的活动
- 城市分布：按当前“地区范围”逐城统计
- 最值得优先看：活动名1；活动名2；活动名3

如果没有活动：

- 时间范围：...
- 本次找到：0 场
- 结论：两周内暂无符合条件的粤港澳纯正硬核技术活动。

## 2. 快速卡片

```md
## 活动 N｜英文名或原名
中文解释：一句中文解释这场活动是干什么的。
- 时间：YYYY-MM-DD HH:MM
- 城市：
- 主题：
- 值不值得去：值得 / 一般 / 不建议
- 一句话理由：
- 链接：
```

## 3. 完整档案

```md
## 活动 N｜活动名称
- 所在城市：
- 精确活动时间：YYYY-MM-DD HH:MM
- 活动总时长：
- 详细地点：
- 核心技术主题：
- 我能学到什么：
- 含金量评级：高 / 中
- 是否纯正硬核技术活动：是
- 信息来源：
- 公众号专属信息：不适用 / 公众号名+发布时间+推文标题
- 推文或原文链接：
- 报名链接：
- 深大地铁站出发交通建议：
- 综合交通方案+预计耗时+费用：
- 往返全程总耗时：
- 是否值得专程前往：值得 / 一般 / 不建议
- 理由：
- 免费/付费及费用：
- 参与门槛+能否旁听+是否限名额：
```

每个活动之间用 `---` 分隔。

## 4. 候补链接

```md
# 候补链接

## 核心城市候补（可自行判断）
- 活动名｜城市｜未入选原因关键词｜链接
- 活动名｜城市｜未入选原因关键词｜链接
```

如果有其他城市候补，也用同样短格式，不展开长解释。

## 5. 备注

```md
# 备注
- 已剔除的典型原因：商业宣讲 / 时间地点不明 / 主题不够硬核 / 不在时间窗内 / 非公开活动
- 交通时间与费用：如无官方精确数据，可基于公开交通信息估算，并明确标注“估算”
```
